# plg_content_jowebpreview 1.0.1
## Quick Start

Download <a href="https://github.com/JLTRY/plg_content_jowebpreview/releases/latest" target="_blank">latest version</a> of plugin

## Requirements

Joomla 5.0+ 

## Features
Joomla plugin that allows to insert a web page preview in articles

Syntax: insert ```{webpreview url=url}``` in an article
